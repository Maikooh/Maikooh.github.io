Développement web statique

Ce projet, réalisé durant ma première année de Master à l'Université de Tours, avait pour objectif de mettre en pratique mes compétences en HTML5, CSS3 ainsi qu'en gestion de version avec Git et GitHub.

Dans le cadre de ce travail, j'ai développé un site web entièrement statique, sans JavaScript, en respectant les bonnes pratiques de structuration HTML, de mise en page CSS (Flexbox, Grid, responsive design) et de versionnement.

Le thème étant libre, j'ai choisi de consacrer celui-ci à l'artiste Lorène Monciardini (Lomia)

Ce projet m'a permis de consolider mes bases en développement web tout en produisant un site complet, organisé et hébergé via GitHub Pages.
